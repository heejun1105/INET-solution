================================================================================
INET 프로젝트 전달 파일
================================================================================

이 폴더에는 의뢰자께 전달할 **배포 가이드와 핵심 파일**이 포함되어 있습니다.

⚠️ 중요: 이 폴더는 배포 가이드만 포함합니다.
전체 프로젝트 소스 코드는 별도로 압축하여 전달하세요!

================================================================================
📦 포함된 파일
================================================================================

1. Docker 관련
   - Dockerfile
   - docker-compose.yml.example
   - .dockerignore

2. 문서
   - docs/DEPLOYMENT_GUIDE.md (가장 중요!)
   - docs/HANDOVER_CHECKLIST.md
   - 기타 참고 문서들

================================================================================
🚀 전달 방법
================================================================================

1. 전체 프로젝트 압축 (소스 코드, 마이그레이션 파일 모두 포함)
2. 이 delivery 폴더도 함께 전달 (배포 가이드 포함)

================================================================================
🚀 빠른 시작
================================================================================

1. docker-compose.yml.example을 docker-compose.yml로 복사
2. docker-compose.yml에서 DB 비밀번호 설정
3. docker-compose up -d --build 실행

자세한 내용은 docs/DEPLOYMENT_GUIDE.md를 참고하세요.

================================================================================
⚠️ 중요 사항
================================================================================

- 기본 관리자 계정: admin / admin123! (운영에서는 반드시 변경!)
- DB 비밀번호는 환경변수로 설정하세요
- Java 버전 걱정 없습니다 (Docker 이미지에 포함됨)
- DB 마이그레이션은 자동으로 실행됩니다 (전체 프로젝트에 포함됨)

================================================================================
